# 🎯 Raccoon Tools - Granatträning  

Ett **CS2 `.cfg`-script** för att ladda en **granatövningsserver** snabbt och enkelt.

## 📌 Funktion  
- Oändliga granater  
- Respawn för T & CT  
- Köp överallt & obegränsad tid

Lägg `pracc.cfg` i **CS2/cfg**-mappen och kör:
